Question 1

24 Mar 2015

Question: In the $xy$-plane, the curve with parametric equations $x=\cos t$ and $y=\sin t$, $\,0 \leq t \leq \pi$, has length

$A)\ 3$

$B)\ \pi$

$C)\ 3\pi$

$D)\ \frac{3}{2}$

$E)\ \frac{\pi}{2}$

Answer: To solve this problem, we notice that this is the equation of a half-circle with radius 1. The circumference of circle is, of course, $2\pi r$, so the circumference of a half-circle of radius 1 is just $\pi$.

An alternative way to solve this problem is to calculate the arc length. If the function was described as a function of $x$, $f(x)$, then the arc length $s$ of the function from $a$ to $b$ could be found as

$$s = \int_{a}^{b} \sqrt{1 + \left(\frac{d}{dx}f(x)\right)^2}\,dx.$$

However, we are working with parametric equations, so the arc length can be described as

$$s = \int_{a}^{b} \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2}\,dt.$$

For this problem, we find:
$$s = \int_{0}^{\pi} \sqrt{\left(\frac{d}{dt}\cos t\right)^2 + \left(\frac{d}{dt}\sin t\right)^2}\,dt$$ $$s = \int_{0}^{\pi} \sqrt{\left( -\sin t\right)^2 + \left( \cos t\right)^2}\,dt$$ $$s = \int_{0}^{\pi} \sqrt{1}\,dt$$ $$s = \int_{0}^{\pi} dt$$ $$s = \pi - 0 = \pi$$

So, the answer is $B$.
