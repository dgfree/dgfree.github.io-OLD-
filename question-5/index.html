Question 5

28 Mar 2015

Question: Suppose $b$ is a real number and $f(x)=3x^2 + bx + 12$ defines a function on the real line, part of which is graphed below. Then $f(5) =$

$A)\,15$

$B)\,27$

$C)\,67$

$D)\,72$

$E)\,87$

Answer: From the graph, we can tell that $f(2) = 0$. Thus,

$$f(2) = 3(2)^2 + b(2) + 12 = 0$$ $$\implies b = -12$$

Plugging $b$ into the original equation and setting $x=5$,

$$f(5) = 3(5)^2 - 12(5) + 12 = 27.$$

Thus, the answer is $B$.
